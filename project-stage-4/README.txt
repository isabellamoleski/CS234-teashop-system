Running the CS-234 Tea Shop System:
-Download the main github project
-ALL information you need is in CS234-teashop-system/project-stage-4/executable
-Create a folder for the system and copy over everything in the CS234_TeaShop folder
-MAKE SURE your folder has: the lib file, the executable jar file, and 
	the .csv files for sales, items, employees, and customers.
-Double click on the executable .jar file
-There you go!
