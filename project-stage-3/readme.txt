How to compile and run Project Stage 3 of the CS234-teashop-system
- Download and save the project stage 3 folder to your PC
- Open Command Prompt
- In Command prompt, navigate to the place you saved the project stage 3 folder
- Type the following. Javac will compile your program, and the java command will run it. 
        javac Menuu.java
        java Menuu
- Your program should run!
